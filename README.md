# Kiro

A collection of custom Kiro Powers and tools.

## Powers

### migrate-to-kiro-powers
Migrate Claude Code Skills into Kiro's power ecosystem. Paste a GitHub link to a Claude Skill and get a ready-to-use Kiro Power.

### mcp-builder
Guide for creating high-quality MCP servers that enable LLMs to interact with external services. Covers TypeScript and Python implementations with best practices, tool design, and evaluation creation.

## Installation

In Kiro: Powers panel → Add Custom Power → Local Directory → point to the specific power folder:

```
/path/to/Kiro/powers/migrate-to-kiro-powers
/path/to/Kiro/powers/mcp-builder
```
